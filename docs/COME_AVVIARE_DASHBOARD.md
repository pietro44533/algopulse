# Come avviare la Dashboard

1. **Attiva l'ambiente virtuale**:
   ```bash
   source venv/bin/activate
   ```

2. **Avvia il server Flask**:
   ```bash
   flask run
   ```

3. **Apri il browser su**:
   [http://localhost:5000](http://localhost:5000)

> Se il comando `flask` non funziona, prova:
> ```bash
> python3 -m flask run
> ```
